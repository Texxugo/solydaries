# Permitir criação e Validação de Organização

## What to build

Permitir que uma **Pessoa** crie uma **Organização** interna como **Representante da Organização** inicial e solicite **Validação de Organização** com documentos privados.

## Acceptance criteria

- [x] Uma Pessoa autenticada consegue criar uma Organização interna.
- [x] A Pessoa criadora vira Representante da Organização inicial.
- [x] Uma Organização não validada não aparece publicamente.
- [x] O Representante consegue enviar solicitação de Validação de Organização.
- [x] Administradores conseguem aprovar ou rejeitar a solicitação.
- [x] Uma Organização aprovada recebe Selo de Validação e pode ter Página pública.
- [x] Documentos de Organização seguem as mesmas regras de arquivo privado.
