# Revisar e publicar Campanhas

## What to build

Implementar a **Revisão de Campanha** para que Campanhas pendentes sejam aprovadas, rejeitadas, publicadas, suspensas ou encerradas conforme decisão de moderação.

## Acceptance criteria

- [x] Um Criador consegue enviar Campanha para revisão.
- [x] Moderadores veem Campanhas pendentes na fila de revisão.
- [x] Moderadores conseguem aprovar Campanha e torná-la pública.
- [x] Moderadores conseguem rejeitar Campanha com motivo.
- [x] Moderadores conseguem suspender Campanha publicada com motivo.
- [x] Responsáveis autorizados conseguem encerrar Campanha.
- [x] Decisões negativas geram notificação interna e registro de auditoria.
