# Adicionar localização de Campanha com Leaflet/OpenStreetMap

## What to build

Adicionar localização de **Campanha** com localidade pública, seleção manual de alfinete usando Leaflet/OpenStreetMap, localização aproximada pública e ponto logístico privado.

## Acceptance criteria

- [x] O Criador seleciona manualmente um pin no mapa ao configurar a Campanha.
- [x] A Campanha salva coordenadas para exibição no mapa.
- [x] A Campanha registra Localidade da Campanha para busca e contexto público.
- [x] O Ponto Logístico da Campanha fica separado da localização pública.
- [x] O pin público evita expor endereço exato por padrão.
- [x] O Moderador consegue ver a localização durante revisão.
