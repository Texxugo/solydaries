# Registrar Oferta de Apoio como Doador

## What to build

Permitir que um **Doador** registre **Oferta de Apoio** para uma Campanha publicada, cobrindo doação de item e voluntariado. (Doação financeira está fora do escopo do MVP.)

## Acceptance criteria

- [x] Visitantes não autenticados não conseguem registrar Apoio.
- [x] Doadores conseguem registrar Oferta de Apoio em Campanha publicada.
- [x] Doação de item coleta item, quantidade, condição, preferência de coordenação e contato.
- [x] Voluntariado coleta disponibilidade, tipo de ajuda, observação e contato.
- [x] O Doador pode escolher anonimato público.
- [x] Detalhes da oferta não aparecem publicamente.
