# Implementar papéis Administrador e Moderador com autorização

## What to build

Adicionar papéis de **Administrador** e **Moderador** com autorização em áreas protegidas, mantendo a separação entre administração de validações e moderação de conteúdo.

## Acceptance criteria

- [x] O sistema representa os papéis Donor, Administrator e Moderator.
- [x] Um Administrador consegue acessar uma área administrativa.
- [x] Um Moderador consegue acessar uma área de moderação.
- [x] Uma Conta sem papel adequado não consegue acessar áreas administrativas ou de moderação.
- [x] Existe forma de criar ou popular uma Conta administrativa inicial para demonstração.
