# Criar base fullstack com login de Pessoa

## What to build

Criar a base fullstack da Solydaries com cadastro e login de **Pessoa** por e-mail e senha, aceite de termos, sessão protegida e estrutura inicial em pt-BR.

## Acceptance criteria

- [x] Uma Pessoa consegue criar uma Conta usando e-mail e senha.
- [x] O cadastro exige aceite dos termos e política de privacidade.
- [x] Uma Pessoa autenticada consegue acessar uma área protegida.
- [x] Uma Pessoa não autenticada é redirecionada ao tentar acessar área protegida.
- [x] A interface inicial usa português do Brasil.

## Blocked by

None - can start immediately
